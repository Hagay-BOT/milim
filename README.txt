אוצר מילים — פסיכומטרי (עברית)
================================

אפליקציית לימוד אוצר מילים לפסיכומטרי, מבוססת על 10 היחידות של
"פסיכומטרי קמפוס — המילון הרשמי" (מ-Quizlet) + מילים אישיות.

קבצים
-----
index.html   — האפליקציה (מסך בית, יחידות, תרגול, סטטיסטיקה)
app.js       — הלוגיקה
data.js      — כל המילים (10 יחידות + אישיות). נוצר אוטומטית.
manifest.webmanifest, sw.js, icon-*.png — תשתית PWA לטלפון (אופליין)
יחידות/יחידה N/words.txt — קובץ מילים לכל יחידה (מקור קריא)
DEPLOY.md    — איך להעלות לאוויר ולהתקין בטלפון

מה יש באפליקציה
---------------
• בחירת יחידה 1–10, "כל המאגר", או "אקראי" (ערבוב מכל היחידות).
• לכל תחום: גרף התקדמות (חזקות/לחיזוק/חדשות) + סטטיסטיקת חוזק לכל מילה (0–5).
• שני מצבי תרגול:
    - "תרגל חלשות" — מילים שכבר תרגלת וחלש בהן (לא כולל חדשות).
    - "מילים חדשות" — מילים שעוד לא תרגלת, ללמוד לראשונה.
  שניהם זמינים ברמת יחידה וברמה כללית.
• תרגול: רואים את הפירוש, כותבים את המילה. בדיקה גמישה (מתעלמת מניקוד/רווחים,
  מקבלת חלק מהצורות). Enter מגיש וממשיך.
• אסוציאציה אישית לכל מילה + כפתור "💡 רמז" בזמן השאלה.
• "אני מכיר" — מחיקת מילה בזמן התרגול.
• ניהול מילים (חיפוש, מחיקה, שחזור) והוספת מילים משלך.
• הכל נשמר מקומית בדפדפן/טלפון: התקדמות, אסוציאציות, מחיקות, הוספות.

עדכון הנתונים מ-Quizlet
-----------------------
הנתונים חולצו מ-Quizlet (2049 מילים). כדי לרענן: להריץ מחדש את סקריפט
החילוץ (scratchpad/gen_from_json.py על קובץ hw_units.json) — הוא מייצר את
data.js ואת קובצי היחידות.
